    bool isLinkLocal() const¶

    Returns true if address is link local unicast. 

                    IPv4 link local addresses are in the 169.254.0.0/16 range, according to RFC 3927.
                IPv6 link local addresses have 1111 1110 10 as the first 10 bits, followed by 54 zeros.
    Returns:

    True if address is link local unicast

    bool isBroadcast() const¶

    Returns true if IP address is broadcast. 

                    Only IPv4 addresses can be broadcast addresses. In a broadcast address, all bits are one.
                For an IPv6 address, returns always false.

    Address::Parse("255.255.255.255").isBroadcast() ; // True
Address::Parse("::").isBroadcast() ; // False
    Returns:

    True if address is broadcast

    bool isUnicast() const¶

    Returns true if address is unicast. 

                    An address is unicast if it is neither a wildcard, broadcast or multicast address.
    Returns:

    True if address is unicast

    bool isMulticast() const¶

    Returns true if address is multicast. 

                    IPv4 multicast addresses are in the 224.0.0.0 to 239.255.255.255 range (the first four bits
                have the value 1110). IPv6 multicast addresses are in the FFxx:x:x:x:x:x:x:x range.
    Returns:

    True if address is multicast

    bool isGlobalMulticast() const¶

    Returns true if address is global multicast. 

                    For IPv4, global multicast addresses are in the 224.0.1.0 to 238.255.255.255 range.
                For IPv6, global multicast addresses are in the FFxF:x:x:x:x:x:x:x range.
    Returns:

    True if address is global multicast

    bool isLinkLocalMulticast() const¶

    Returns true if address is link-local multicast. 

                    For IPv4, link-local multicast addresses are in the 224.0.0.0/24 range.
                For IPv6, link-local multicast addresses are in the FFx2:x:x:x:x:x:x:x range.
    Returns:

    True if address is link-local multicast

    bool isNodeLocalMulticast() const¶

    Returns true if address is node-local multicast. 

                    IPv4 does not support node-local addresses, thus the result is always false for an IPv4
                address. For IPv6, node-local multicast addresses are in the FFx1:x:x:x:x:x:x:x range.
    Returns:

    True if address is node-local multicast

    bool isOrgLocalMulticast() const¶

    Returns true if address is organization-local multicast. 

                    For IPv4, organization-local multicast addresses are in the 239.192.0.0/16 range.
                For IPv6, organization-local multicast addresses are in the FFx8:x:x:x:x:x:x:x range.
    Returns:

    True if address is organization-local multicast

    bool isSiteLocal() const¶

    Returns true if address is site local unicast. 

                    IPv4 site local addresses are in on of the 10.0.0.0/24, 192.168.0.0/16 or 172.16.0.0 to
                172.31.255.255 ranges. Originally, IPv6 site-local addresses had FEC0/10 (1111 1110 11)
                prefix (RFC 4291), followed by 38 zeros. Interfaces using this mask are supported, but
                obsolete; RFC 4193 prescribes fc00::/7 (1111 110) as local unicast prefix.
    Returns:

    True if address is site local unicast

    bool isSiteLocalMulticast() const¶

    Returns true if address is site-local multicast. 

                    For IPv4, site local multicast addresses are in the 239.255.0.0/16 range.
                For IPv6, site-local multicast addresses are in the FFx5:x:x:x:x:x:x:x range.
    Returns:

    True if address is site-local multicast

    bool isWellKnownMulticast() const¶

    Returns true if address is well-known multicast. 

                    For IPv4, well-known multicast addresses are in the 224.0.0.0/8 range.
                For IPv6, well-known multicast addresses are in the FF0x:x:x:x:x:x:x:x range.
    Returns:

    True if address is well-known multicast

    bool isIPv4Compatible() const¶

    Returns true if address is IPv4 compatible. 

                    For IPv4 addresses, this is always true.
                For IPv6, the address must be in the ::x:x range (the first 96 bits are zero).
    Returns:

    True if address is IPv4 compatible

    bool isIPv4Mapped() const¶

    Returns true if address is an IPv4 mapped IPv6 address. 

                    For IPv4 addresses, this is always true.
                For IPv6, the address must be in the ::FFFF:x:x range.
    Returns:

    True if address is an IPv4 mapped IPv6 address.
